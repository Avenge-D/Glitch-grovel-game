# Glitch-grovel-game
My latest release, Glitch Grovel, is a web-based, 16:9 arcade adventure built entirely with Vanilla JavaScript and HTML5 Canvas. It serves as a masterclass in scaling difficulty and atmospheric world-building.
50 Levels of Chaos: The game features a brutal difficulty ramp, progressing from introductory platforming to "Nightmare" and "Extreme" modes with only a 1% survival chance.  
Dynamic Environments: Implemented a sophisticated day/night cycle that toggles every five levels. The engine renders parallax clouds, a flickering sun with ray-tracing effects, and deep-space nebula blobs for night missions.  Sophisticated Trap Systems: I designed a diverse library of hazards, including vanishing platforms, heat-sensitive "hot floors," collapsing tiles, and proximity-based "spikepop" traps.  
High-Fidelity Procedural Animation: The protagonist is a realistically rendered lion featuring multi-layered mane physics, breathing cycles, and a dynamic walking gait that adapts to player input.  
Technical ExpertiseCore Languages: Advanced JavaScript (ES6+), HTML5, and CSS3.  Graphics & Physics: Expert-level HTML5 Canvas API for high-performance 2D rendering and custom collision detection. 
Mobile-First UX: Fully responsive UI/UX design, including orientation-aware layouts and custom-mapped touch controls.  The Design PhilosophyI believe games should be more than just entertainment—they should be a challenge of wit.
By integrating humor into the failure state (such as custom "Troll Messages" on death screens), I turn player frustration into engagement.
